# Contributing to OCTA AI

Thank you for your interest in OCTA AI!

## Guidelines

- All contributions must be for authorized security research
- Never add code that enables unauthorized access
- Test all changes before submitting
- Follow Python PEP8 style guide

## Adding New Tools

1. Add tool to appropriate module
2. Import from keys.py for AI calls
3. Import from memory.py to save results
4. Import from outputs.py for reports
5. Update README.md

## Reporting Issues

Open a GitHub issue with:
- OS version
- Python version
- Error message
- Steps to reproduce
